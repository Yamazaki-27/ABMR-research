# CLAUDE.md

## プロジェクト概要

AMR（自律搬送ロボット）の自社開発・基礎研究プロジェクト。
工場内の重量物搬送を自動化するAMRを、試作・実験を繰り返しながら開発中。
技術部としてGitHubを実際の開発に活用する、社内初のトライ。

## ファイル構成

```
ABMR-research/
├── firmware/      ← ロボット制御プログラム（IDECコントローラー等）
├── sensor/        ← センサー関連プログラム・設定
├── docs/
│   ├── 議事録/    ← 社長・役員プレゼン記録等
│   ├── 仕様書/    ← IDEC・DMR-SEERからの資料等
│   └── メモ/      ← 各メンバーの個人フォルダ
├── Images/        ← 写真（500KB以下に縮小してから追加）
└── README.md
```

## 運用ルール

- `firmware/` `sensor/` への変更はブランチを切ってからPR経由でマージ
- `docs/`（議事録・仕様書・メモ）は競合リスクが低いため、`main` に直接pushでOK
- 写真は必ず **500KB以下** に縮小してから追加する
- 動画は外部リンク（YouTube・Googleドライブ等）で共有する

## コミットメッセージルール

```
feat: ライントレースの速度制御を追加
fix: 右旋回時のモーターズレを修正
docs: 実験記録2026-05-18を追加
```

`修正した` / `test` / `aaa` などは不可。何をしたか一目でわかるメッセージを書くこと。
